This folder contains asset sources. These are compiled by the engine and placed in content.


Things that belong in this folder:

	Map vmap files
	Raw textures (tga, png, psd etc)
	Model + Animation source files (fbx, vmdl etc)


Things that don't belong here:

	UI files
	Text files that you want to load



People don't need this folder to play your game. When you come to distribute it you can delete this folder.
